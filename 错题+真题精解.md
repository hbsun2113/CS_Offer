#### [Q 1](http://www.cnblogs.com/qlwy/archive/2011/08/25/2153584.html) :

```
题目：
    下列程序的输出结果：
    #include <iostream>
    using namespace std;
    class A{
    public:
        void print(){
            cout << "A:print()";
        }
    };
    class B:private A{
    public:
        void print(){
          cout << "B:print()";
        }
    };
    class C:public B{
    public:
       void print(){
            A::print();
        }
    };
    int main(){
        C b;
        b.print();
    }

答案:
    编译出错

解答:
    考察C++继承问题。
    1. 类B私有继承类A。
    2. 私有继承：类A的公有成员和保护成员都作为类B的私有成员，并且不能被类B的子类（如类C）所访问。
```
